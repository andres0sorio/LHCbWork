How to produce the documentation using doxygen

you are already under CPStudies/doc
there is a file call config
run from the prompt:

% doxygen config

this will produce documentation in html and latex

doc/html: open in your browser index.html
doc/latex: you will need to run the Makefile
% make
% make pdf

open the refman.pdf file

feel free to make change the config file
